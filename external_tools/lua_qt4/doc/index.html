lua_qt - Qt bindings for the Lua language.

lua_qt is a set of bindings to use the Qt toolkit with the Lua language.
It was done using qt 4.0.1 on Debian.

lua_qt requires tolua++ version 1.0.6 or later

Latest version is 0.0.2a, and supports Qt 4.0.1. Older versions:
